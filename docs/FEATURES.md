# FEATURES — INNOCEAN Benchmark

## 페이지 구성

1. **벤치마크 조회** — 업종별 광고 KPI 벤치마크 검색 + 필터.
2. **AI 분석 채팅** — 자연어로 질문 → 벤치마크 데이터 기반 응답.
3. **Summary 차트** — 업종별 / 매체별 KPI 분포 시각화.

---

## 1. 벤치마크 조회

### 필터

- **업종**: 자동차 / 금융 / 화장품 / F&B / 전자 / 패션 등.
- **매체**: Google / Meta / 네이버 / 카카오 / YouTube / 인스타.
- **기간**: 2024 / 2025 / 2026 (분기 단위).
- **KPI**: CPM / CPC / CTR / VTR / CVR / ROAS.

### 결과 표시

- 표 형식 (업종 × 매체 × KPI).
- 평균 / 중앙값 / 상위 25% / 상위 10% 4분위 표시.
- 캠페인 비교 가능.

---

## 2. AI 분석 채팅

### 입력 패턴 (rule-based)

| 의도 | 키워드 | 동작 |
|---|---|---|
| 업종 조회 | "[업종] 평균 / 벤치마크" | 해당 업종 KPI 평균 + 중앙값 응답 |
| 매체 비교 | "[매체A] vs [매체B]" | 두 매체 KPI 비교 차트 + 텍스트 |
| 이상치 탐지 | "이상치 / 특이값" | 평균 대비 ±2σ 벗어난 케이스 |
| 트렌드 | "분기별 / 추이" | 시계열 차트 응답 |

### Phase 2 예정

- LLM 연동 (Gemini Flash) — 자연어 이해 강화.
- 캠페인 자동 추천.
